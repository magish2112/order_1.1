# Админ-панель

Административная панель для управления контентом сайта ремонтно-строительной компании.

## Технологический стек

- **React** 18.x
- **Vite** 5.x
- **TypeScript** 5.x
- **Ant Design** 5.x - UI компоненты
- **TanStack Query** - управление состоянием сервера
- **TanStack Table** - таблицы с сортировкой/фильтрацией
- **React Router** 6.x - маршрутизация
- **React Hook Form** + **Zod** - формы и валидация
- **TipTap** - WYSIWYG редактор
- **Zustand** - управление состоянием клиента

## Установка

```bash
npm install
```

## Запуск

### Разработка
```bash
npm run dev
```

Приложение будет доступно по адресу: http://localhost:3001

### Продакшн
```bash
npm run build
npm run preview
```

## Переменные окружения

Создайте файл `.env`:

```
VITE_API_URL=http://localhost:4000/api/v1
```

## Функционал

### ✅ Полностью реализовано

- ✅ **Аутентификация** - login, logout, автоматическое обновление токенов
- ✅ **Dashboard** - статистика заявок, проектов, статей, сотрудников
- ✅ **Услуги** - CRUD категорий и услуг с поиском и фильтрацией
- ✅ **Проекты** - CRUD с загрузкой галерей (до/после/дизайн), связь с услугами
- ✅ **Статьи** - CRUD с WYSIWYG редактором (TipTap), публикация
- ✅ **Заявки** - список, фильтрация, изменение статуса, заметки
- ✅ **Сотрудники** - CRUD с загрузкой фото, группировка по отделам
- ✅ **Отзывы** - модерация, одобрение, связь с проектами
- ✅ **Вакансии** - CRUD вакансий
- ✅ **FAQ** - CRUD вопросов-ответов с категоризацией
- ✅ **Медиатека** - загрузка файлов, просмотр, удаление, миниатюры
- ✅ **Настройки** - контакты, соцсети, калькулятор, SEO

## API Endpoints

Все запросы идут на `/api/v1/admin/*` (кроме auth, который использует `/api/v1/admin/auth/*`).

## Авторизация

Используется JWT с access и refresh токенами. Токены автоматически обновляются при истечении access token.
